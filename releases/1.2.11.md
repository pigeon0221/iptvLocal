## What's New

- Hosted Feeds: live channels now play reliably and start quickly in Jellyfin / Plex / Emby — fixing both a playback failure and the slow first-play delay.
- Hosted Feeds: the TV guide updates more reliably.
- Live streams: a channel that drops to a backup source now returns to its main source on its own, so you're not left on a lower-quality backup.
- Custom sources: fixed a refresh that could leave a source showing no channels.
- Dashboard: fixed the login screen getting stuck on "Initializing" when accessed through a reverse proxy.

## Docker

```bash
docker pull rebeliptv/iptv:1.2.11
```

### Docker Compose

```yaml
services:
  iptv:
    image: rebeliptv/iptv:1.2.11
    hostname: iptv
    container_name: iptv
    ports:
      - "8080:8080"
    # Environment variables are OPTIONAL — defaults work out of the box.
    # Uncomment any you need:
    # environment:
    #   TZ: "America/New_York"   # set to YOUR timezone, e.g. Europe/London, Etc/UTC
    volumes:
      - iptv-data:/app/data
    dns:
      - 8.8.8.8
      - 4.2.2.2
      - 1.1.1.1
    stop_grace_period: 30s
    restart: unless-stopped

volumes:
  iptv-data:
```

### Docker Run

```bash
docker run -d \
  --name iptv \
  -p 8080:8080 \
  -v iptv-data:/app/data \
  --dns 8.8.8.8 \
  --dns 4.2.2.2 \
  --dns 1.1.1.1 \
  --restart unless-stopped \
  rebeliptv/iptv:1.2.11

# Environment variables are optional (defaults work). Add -e flags as needed, e.g.:
#   -e TZ=America/New_York   (set to YOUR timezone — e.g. Europe/London, Etc/UTC)
```

### Updating

```bash
docker pull rebeliptv/iptv:1.2.11
docker compose up -d
```

## Endpoints

| URL | Description |
|-----|-------------|
| `http://<server>:8080/playlist?key=YOUR_KEY` | M3U playlist |
| `http://<server>:8080/epg?key=YOUR_KEY` | EPG guide (XML) |
| `http://<server>:8080/epg.gz?key=YOUR_KEY` | EPG guide (gzip) |
| `http://<server>:8080` | Web dashboard |

> **Note:** The `?key=` parameter is only required if an API key has been generated in the web dashboard under **Settings**.
